# keystonejs-playground

Learn and experiment with [KeystoneJS](http://keystonejs.com/)

## Install and Run

```sh
$ npm install

# create dir for mongodb files
$ mkdir -p data/db

# start mongodb
$ mongod --dbpath data/db &

# start webapp
$ node keystone.js

# visit http://localhost:3000/
```
